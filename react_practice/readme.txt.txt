this repository fo starting with react Js and building some inuitive components with it. Suitable for beginers.

1- To install react:	#npx create-react-app my-app % this ensure installing the last version of react

***alternatively:	#npm i -g create-react-app@1.5.2 %to install specific version	#npm uninstall -g create-react-app   %To uninstall it

2- Install the following extensions from Visual studio code (VSC):	#Simple React snippets & prettier

3- create a directory where you will work on react
Example:	#create-react-app myapp

	navigate to the created directory:	#cd myapp
						#npm start
